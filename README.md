## Okashi

### Prerequirements

  -  node

### Setup

```
$ npm install -g cordova ionic

```

### To start emulator:

```
$ ionic platform add ios
$ ionic build ios
$ ionic emulate ios
```

### To start server on browser

```
$ ionic serve

```

### e2e Test

```
$ npm install -g protractor
$ webdriver-manager update
```

To test:
```
webdriver-manager start
```
